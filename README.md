# EyeVox

EyeVox is an innovative application that combines gaze control and voice commands to enhance accessibility and interaction. It leverages cutting-edge technologies such as dlib, OpenCV, and speech recognition to provide seamless user experiences.

## Features

- **Gaze Control**: Move the cursor using eye movements.
- **Voice Commands**: Execute actions such as typing and opening applications with your voice.
- **Real-Time Interaction**: Works in real-time for efficient and natural control.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/EyeVox.git
   cd EyeVox
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Download the pre-trained model and place it in the `models/` directory.

## Usage

Run the main script to start the application:
```bash
python src/main.py
```
